Large eddy simulations of frazil ice advection, formation, and melting in the turbulent upper ocean, built on [Oceananigans](https://github.com/CliMA/Oceananigans.jl).

## Installation

`SnowingOcean` is not registered. First [install Julia](https://julialang.org/downloads/), then clone this repository. The package's own dependencies are instantiated with

```julia
julia> using Pkg; Pkg.activate("."); Pkg.instantiate()
```

You can then load the package's utilities with `using SnowingOcean`.

## The `SnowingOcean` module

The `src/` module collects the physics shared across the examples:

- `monochromatic_stokes_drift(; amplitude, wavelength, gravitational_acceleration=9.81)` builds a `UniformStokesDrift` for a deep-water surface gravity wave.
- `rising_tracer_forcing(grid, w)` builds an `AdvectiveForcing` that advects a tracer upward at constant velocity `w`, with impenetrable top and bottom boundaries so total tracer is conserved.
- `linear_freezing_temperature(S, z; slope=0.0545, depth_coefficient=7.9e-4)` returns the linear-liquidus approximation of the seawater freezing temperature.

## Examples

The scripts in `examples/` use these utilities together with `GLMakie` for plotting and `GibbsSeaWater` for TEOS-10 reference values. They have their own environment:

```julia
julia> using Pkg
julia> Pkg.activate("examples")
julia> Pkg.develop(path=".")   # make the SnowingOcean source available
julia> Pkg.instantiate()
```

Then run, for example,

```julia
julia> include("examples/langmuir_with_teos10.jl")
```

- `simple_buoyant_tracer.jl` — Langmuir turbulence with a `BuoyancyTracer` plus a passive rising tracer (runs on a `GPU`).
- `langmuir_with_teos10.jl` — the same setup using the TEOS-10 equation of state for buoyancy.
- `freezing_temperature.jl`, `linear_liquidus.jl` — analysis of the seawater freezing temperature and its linear approximation.

## Freezing temperature and thermal expansion coefficient as a function of salinity and depth

This plot is generated by running `examples/freezing_temperature.jl`:

<img width="1191" alt="image" src="https://github.com/glwagner/SnowingOcean/assets/15271942/8f806c7f-7678-4c08-8c80-57ee94d2883b">

Note that the freezing temperature of seawater decreases with depth. This means that frazil ice advected to depth will melt.
Moreover, the thermal expansion coefficient of seawater is positive for moderately saline waters at the freezing temperature.
This means that cooling due to latent heat fluxes during frazil ice melting at depth makes the ocean more dense.
