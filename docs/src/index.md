# SnowingOcean.jl

SnowingOcean.jl provides large-eddy simulations of frazil ice and ice-shelf
cavity processes in the turbulent upper ocean, built on
[Oceananigans](https://github.com/CliMA/Oceananigans.jl).

## Installation

SnowingOcean.jl is not yet registered. To use it, clone the repository and
develop it into your Julia environment:

```julia
using Pkg
Pkg.develop(PackageSpec(url="https://github.com/NumericalEarth/SnowingOcean.jl.git"))
```

Alternatively, clone the repository manually and then run:

```julia
using Pkg
Pkg.develop(PackageSpec(path="/path/to/SnowingOcean.jl"))
```
